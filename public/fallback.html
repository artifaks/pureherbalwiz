Troubleshooting the White Screen Issue

If you're seeing a white screen on your main application, this fallback page can help diagnose and fix the issue.

Common Causes:

  • Missing environment variables
  • MIME type configuration issues
  • JavaScript module loading problems
  • Client-side routing configuration

Environment Variables Check

Make sure these environment variables are set in your Netlify dashboard:

  • VITE_SUPABASE_URL - Your Supabase project URL
  • VITE_SUPABASE_ANON_KEY - Your Supabase Anon Key
  • VITE_STRIPE_PUBLIC_KEY - Your Stripe Public Key

MIME Type Configuration

Your site should have proper MIME type configuration in the netlify.toml file:

[[headers]]
  for = "/*.js"
  [headers.values]
    Content-Type = "application/javascript; charset=utf-8"

[[headers]]
  for = "/*.mjs"
  [headers.values]
    Content-Type = "application/javascript; charset=utf-8"

[[headers]]
  for = "/*.css"
  [headers.values]
    Content-Type = "text/css; charset=utf-8"
